One-time setup

Create a file named config.js in the same folder as this page, with your keys:

window.CONFIG = {
  ANTHROPIC_KEY: 'sk-ant-...',
  GITHUB_TOKEN:  'ghp_...',
  GITHUB_REPO:   'tkandhati/AI-Engineer',
  PLAN_START:    '2025-04-01'
};

A template is in config.template.js. Rename it to config.js and fill in values.

GitHub token: Settings → Developer Settings → Personal Access Tokens → repo scope.
config.js is in .gitignore — never pushed.
